ITango
======

ITango is a PyTango CLI based on IPython_. It is designed to be used as an
IPython profile.

ITango is available since PyTango 7.1.2

You can start ITango by typing on the command line:

    #. For ipython <= 0.10::
        
        $ ipython -p tango

    #. For ipython > 0.10::

        $ ipython --profile=tango
